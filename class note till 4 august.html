1. Linear Algebra

1.1 Matrices

Definition: Matrix

A matrix is a rectangular array of numbers arranged in rows and columns. If a matrix has 'm' rows and 'n' columns, it is called an m×n matrix.

Types of Matrices (All Types Required for Homework)

Matrix Type Definition (English) Definition (Bengali)
Row Matrix A matrix with only one row (1×n) সারি ম্যাট্রিক্স
Column Matrix A matrix with only one column (m×1) কলাম ম্যাট্রিক্স
Zero Matrix All elements are zero শূন্য ম্যাট্রিক্স

Trace of a Matrix

For a square matrix A, the trace is the sum of the elements on the main diagonal:

tr(A) = a₁₁ + a₂₂ + ... + aₙₙ

Matrix Multiplication

Example: Matrix Multiplication

Given matrices A (2×3) and B (3×2):

A = [2 3 5]
[3 -1 4]
B = [1 2]
[2 3]
[3 4]

Solution:

A × B = [(2×1 + 3×2 + 5×3) (2×2 + 3×3 + 5×4)]
          [(3×1 + (-1)×2 + 4×3) (3×2 + (-1)×3 + 4×4)]

       = [(2 + 6 + 15) (4 + 9 + 20)]
         [(3 - 2 + 12) (6 - 3 + 16)]

A × B = [23 33]
[13 19]

Matrix Multiplication Rules:

  • For A(m×n) × B(p×q): multiplication is possible only if n = p.
  • The resulting matrix will have dimensions (m×q).
  • Generally, AB ≠ BA (it is not commutative).

2. Analytic and Vector Geometry

2.1 Coordinate Systems

Coordinate Systems

Cartesian Coordinates (কার্তেসীয় স্থানাঙ্ক): A point is represented by an ordered pair (x, y).

Polar Coordinates (পোলার স্থানাঙ্ক): A point is represented by a distance from the origin (r) and an angle (θ).

Coordinate Transformations

Cartesian ↔ Polar Conversion:

x = r cos θ      y = r sin θ

r = √(x² + y²)      θ = tan⁻¹(y/x)

2.2 Transformation of Coordinates

Types of Transformations:

  1. Translation (মূল বিন্দুর স্থানান্তর): Changing the origin's position while keeping axis directions unchanged.
  2. Rotation (অক্ষের আবর্তন): Rotating axes around the origin while keeping it fixed.
  3. Combined: Both translation and rotation.

Theorem 1: Translation of Axes

Statement: If the origin is translated to a new point O'(α, β) while keeping the axis directions unchanged, then:

(a) The new coordinates (x₁, y₁) of a point P(x,y) are: (x₁, y₁) = (x-α, y-β)

(b) The transformed equation of f(x,y) = 0 is: f(x₁+α, y₁+β) = 0

Theorem 2: Rotation of Axes

Statement: If the axes are rotated by an angle θ while keeping the origin fixed, then:

(a) The new coordinates (x₁, y₁) are related to the old coordinates (x, y) by:

x₁ = x cos θ + y sin θ
y₁ = -x sin θ + y cos θ

(b) The old coordinates (x, y) can be expressed in terms of the new coordinates (x₁, y₁) as:

x = x₁cos θ - y₁sin θ
y = x₁sin θ + y₁cos θ

3. Vectors in Rⁿ and Cⁿ

3.1 Vector Notation

Vector in Rⁿ (Real Space)

A vector u in n-dimensional real space is an ordered n-tuple:

u = (u₁, u₂, ..., uₙ) ∈ Rⁿ

Vector in Cⁿ (Complex Space)

A vector u in n-dimensional complex space where each component uᵢ is a complex number:

u = (u₁, u₂, ..., uₙ) ∈ Cⁿ

3.2 Dot Product (Inner Product)

Dot Product in Rⁿ

For vectors u = (u₁, ..., uₙ) and v = (v₁, ..., vₙ) in Rⁿ:

u · v = u₁v₁ + u₂v₂ + ... + uₙvₙ

Dot Product in Cⁿ

For vectors u, v ∈ Cⁿ:

u · v = u₁v̅₁ + u₂v̅₂ + ... + uₙv̅ₙ

Note: v̅ᵢ represents the complex conjugate of vᵢ.

Commutative Property of Dot Product:

In Real vector spaces (Rⁿ): u · v = v · u (Commutative)

In Complex vector spaces (Cⁿ): u · v ≠ v · u. Instead, u · v = (v · u)̅ (Conjugate symmetric).

3.3 Norm (Length) and Distance

Concept In Rⁿ (Real Space) In Cⁿ (Complex Space)
Norm (Length) ||u||
||u|| = √(u₁² + u₂² + ... + uₙ²)
||u|| = √(|u₁|² + |u₂|² + ... + |uₙ|²)
Distance d(u,v)
d(u,v) = ||u-v|| = √((u₁-v₁)² + ... + (uₙ-vₙ)²)
d(u,v) = ||u-v|| = √(|u₁-v₁|² + ... + |uₙ-vₙ|²)

Example: Distance Calculation in C³

Let u = (i, 1-i, -3+i) and v = (2, 3+i, -i)

Step 1: Find u - v

u-v = (i-2, (1-i)-(3+i), (-3+i)-(-i)) = (-2+i, -2-2i, -3+2i)

Step 2: Calculate the squared magnitude of each component

|-2+i|² = (-2)² + 1² = 5

|-2-2i|² = (-2)² + (-2)² = 8

|-3+2i|² = (-3)² + 2² = 13

Step 3: Sum the squares and take the square root

d(u,v) = ||u-v|| = √(5 + 8 + 13) = √26

5. Functions

Definition: Function

A relationship between two non-empty sets A and B is called a function from A to B if for every element in set A, there exists one and only one image in set B.

f: A → B

Domain (ডোমেন): The set of all possible inputs (Set A). Df = A.

Codomain (কোডোমেন): The set of all possible outputs (Set B). Codf = B.

Range (রেঞ্জ): The set of all actual outputs. The range is a subset of the codomain.

Types of Functions

Function Type Definition
One-to-One (Injective) Function (এক-এক ফাংশন) Every element in the domain has a unique image in the codomain. No two inputs have the same output.
Onto (Surjective) Function (সার্বিক ফাংশন) Every element in the codomain is an image of at least one element from the domain. (Range = Codomain).
Constant Function (ধ্রুবক ফাংশন) Every element in the domain maps to the exact same element in the codomain.
